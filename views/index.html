How to use the Triosis Website

User Routes

  • GET (home): /
  • GET: /users
  • GET: /users/:userId
  • POST: /users
  • PATCH: /users
  • PUT: /users/:userId
  • DELETE: /users/:userId
  • GET: /users/:userId/cart
  • POST: /users/:userId/cart
  • PUT: /users/:userId/cart
  • DELETE: /users/:userId/cart

Product Routes

  • GET: /products
  • GET: /products/:product_id
  • POST: /products
  • PUT: /products/:product_id
  • DELETE: /products/:product_id
